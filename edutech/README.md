# EduTech Dashboard (Peru Market Focus)

## Overview

This dashboard tracks educational technology trends with a focus on the Peru market:
- RSS Feeds (EdSurge, EdTech Magazine, eLearning Industry)
- GitHub Releases (Moodle, OpenEdX)
- Reddit (r/edtech, r/education)
- YouTube (EdTech Hustle)

## Update Schedule

The dashboard updates **every 6 hours** via GitHub Actions.
