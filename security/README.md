# Security Intelligence Dashboard

**Last Updated:** Not yet run (initial setup)

## Overview

This dashboard tracks cybersecurity intelligence and threat information:
- RSS Feeds (Krebs on Security, The Hacker News, Schneier on Security, BleepingComputer)
- GitHub Releases (OWASP Top 10, MITRE ATT&CK, Metasploit)
- Reddit (r/netsec, r/cybersecurity)
- YouTube (SANS Cyber Security, Black Hat)

## Update Schedule

The dashboard updates **every 6 hours** via GitHub Actions.

## Configuration

Data sources: `dashboards/security/sources.yaml`

Manage sources with: `/7f-dashboard-curator`
