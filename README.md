## Configuration

Each dashboard has its own configuration:
- `dashboards/{name}/sources.yaml` - Data source configuration
- `dashboards/{name}/.github/workflows/` - GitHub Actions workflows
- `dashboards/{name}/data/` - Cached data and metadata
- `dashboards/{name}/summaries/` - Weekly AI summaries

---

## Usage

### View Dashboard
Navigate to the dashboard directory and open README.md

### Manage Data Sources
Use the dashboard curator skill:
```
/7f-dashboard-curator
```

### Manual Update
Trigger workflow manually via GitHub Actions UI, or run locally:
```bash
python scripts/update_dashboard.py --config dashboards/{name}/sources.yaml --dashboard {name}
```
