# Fintech Trends Dashboard

**Last Updated:** Not yet run (initial setup)

## Overview

This dashboard tracks the latest fintech trends and developments from multiple sources:
- RSS Feeds (Fintech News, The Financial Brand, Payments Source)
- GitHub Releases (Stripe, Plaid SDKs)
- Reddit (r/fintech)
- YouTube (Fintech Insider)

## Update Schedule

The dashboard updates **every 6 hours** via GitHub Actions.

## Configuration

Data sources: `dashboards/fintech/sources.yaml`

Manage sources with: `/7f-dashboard-curator`
